# MeshSplitter
メッシュを分割するためのEditor拡張です
マテリアル毎に分割する「split by materials」と
Quad（平面）で指定した場所で分割する「split by Quad」の機能があります。
現在は、SkinedMeshRendererのみ分割可能です。

また、分割前のメッシュは、非アクティブ化しているだけで、データ上は残っています。
VRChat等にアップロードする際には、自己判断で削除するなどしてください。
